# admindoorstep

A web-primary Flutter admin dashboard for doorstep operations.

## Web-First Development

This project should be treated as web-primary by default. When making future UI
changes, prefer:

- wide-screen admin layouts first
- browser-friendly navigation and workflows
- responsive behavior that still adapts cleanly to smaller screens

## Run Locally

```bash
flutter pub get
flutter run -d chrome
```

## Build For Web

```bash
flutter build web
```
